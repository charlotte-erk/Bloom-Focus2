A simple, interactive focus timer inspired by the Pomodoro technique. Watch a flower bloom as you work, helping you stay focused and track your time visually.

## Features

- 5-stage flower animation that grows as time passes.
- Preset time buttons: 1 min, 5 min, 15 min, 30 min, 60 min.
- Pause, play, and reset controls.
- Timer completion triggers a gentle ‚Äútwinkle‚Äù sound.
- Clean, minimalistic design with thin sans-serif text.

## How to Use

1. Open `index.html` in your browser.
2. Select a preset time or run the default time.
3. Click **Play** to start the timer.
4. Watch the flower bloom as the timer counts down.
5. Click **Pause** to pause the timer or **Stop** to reset.
